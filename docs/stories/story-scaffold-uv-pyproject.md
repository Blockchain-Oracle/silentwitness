## Acceptance criteria (BDD — machine-verifiable)

```
Given pyproject.toml + uv.lock + ruff.toml + mypy.ini + .python-version exist
When  `uv sync --frozen` runs from a clean checkout
Then  exit code is 0
And   `.venv/` is created
And   `uv pip list` lists `mcp`, `pydantic`, `pydantic-ai`, `typer`, `rich`, `weasyprint`, `mistune`, `httpx`, `spacy`, `volatility3` (NOT `structlog` — dropped per audit Decision A)

Given the dev-deps are installed
When  `uv run ruff check .` and `uv run ruff format --check .` run
Then  both exit 0 on the freshly scaffolded tree

Given mypy.ini sets strict = True
When  `uv run mypy --strict src/` runs
Then  exit code is 0 (empty package init files type-clean)

Given tests/unit/test_scaffold.py exists with the sanity tests
When  `uv run pytest tests/unit/test_scaffold.py -v` runs
Then  exit code is 0
And   exactly 3 tests pass

Given `.python-version` pins 3.12
When  `cat .python-version` runs
Then  output is exactly `3.12`

Given pyproject.toml declares `requires-python = ">=3.12,<3.14"`
When  `grep -E '^requires-python' pyproject.toml` runs
Then  output contains `>=3.12,<3.14`
```
