## Synthetic disclosure

This case is synthetic and is intended solely to make the audit trail
readable without running the system. The evidence file is a 4-byte
placeholder; the tool outputs are hand-crafted to demonstrate the JSONL row
shape, not real Volatility / EZ Tools / EVTX output. Every filename is
suffixed `_EXAMPLE` so the simulation status is obvious to the reader;
nothing in this directory should be interpreted as a real investigation
result. The HMAC ledger key is the literal string
`EXAMPLE-not-a-real-secret-EXAMPLE` — documented here so no reader confuses
it for a real ledger.
